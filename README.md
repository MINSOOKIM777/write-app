## 블로그 글/쇼츠 자동 생성기

로컬에서 입력값을 받아 **블로그 글(1500자 이상)** 과 **쇼츠(45초) 대본**을 버튼으로 생성하는 간단한 도구입니다.

### 실행

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
streamlit run app.py
```

### 사용법

- **블로그 글 작성**: 키워드/주제/톤 등을 입력하고 클릭
- **쇼츠 만들기(45초)**: 동일 주제 기준으로 45초 쇼츠용 **컷 구성 + 나레이션 + 자막** 생성
- 생성 결과는 화면에 표시되고, 동시에 `outputs/` 폴더에 자동 저장됩니다.

### 유튜브 쇼츠 업로드 자동화(선택)

이 도구는 **유튜브 Data API(OAuth)** 를 이용해, 로컬에서 선택한 영상 파일(mp4)을 **내 채널로 업로드**할 수 있습니다.

- **채널별로 필요한 것?**: 보통은 “채널마다 API”가 아니라, **프로필(계정/채널)별로 로그인 토큰을 따로 저장**하는 방식입니다.
- 앱의 **채널 프로필 이름**(예: `main`, `sub`)을 다르게 두면 프로필별 토큰이 분리됩니다.
- OAuth 클라이언트 JSON과 토큰은 로컬 폴더 `./.local_youtube/`에 저장됩니다. (외부 공유 금지)

### 자동화(완전 자동)도 가능해요

- 현재 버전은 **템플릿 기반 자동 생성** + (선택) **LLM API 연동 훅**이 포함되어 있어요.
- `OPENAI_API_KEY` 같은 키를 환경변수로 넣고, `llm.py`의 `generate_with_llm()`를 활성화하면 “입력 → 즉시 고품질 문장 생성”까지 자동화할 수 있습니다.
